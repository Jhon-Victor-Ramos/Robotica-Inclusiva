# Robótica Inclusiva

Bem-vindo ao repositório da disciplina de **Robótica Inclusiva**, cursada durante o 4º período do bacharelado em Ciência da Computação na UNICAP.

## Sobre o Repositório
Este espaço é dedicado ao armazenamento e organização de todos os códigos, esquemas de circuitos e documentações dos projetos práticos desenvolvidos ao longo do semestre. O foco principal da disciplina é utilizar a tecnologia e a robótica para criar soluções de acessibilidade e promover a inclusão.

## Tecnologias Utilizadas
* **Hardware:** Arduino, Sensores, Atuadores e Componentes Eletrônicos diversos.
* **Software/Linguagens:** C/C++ (Arduino IDE).
* **Prototipagem:** Tinkercad

## Como Executar

1. Clone este repositório:
```bash
git clone [https://github.com/Jhon-Victor-Ramos/robotica-inclusiva-unicap.git](https://github.com/Jhon-Victor-Ramos/robotica-inclusiva-unicap.git)
```
2. Abra o arquivo `.ino` desejado na **Arduino IDE**.
3. Conecte sua placa Arduino ao computador.
4. Selecione a placa e a porta correta na IDE (`Ferramentas > Placa` / `Ferramentas > Porta`).
5. Clique em **Carregar** para transferir o código para o microcontrolador.
